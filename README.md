# Advent of Code

My solutions for [Advent of Code](https://adventofcode.com/) challenges across multiple years.

## Structure

```
2025/          - Advent of Code 2025 solutions
```

## About

Advent of Code is an annual set of Christmas-themed programming puzzles that can be solved in any programming language. Each year contains 25 days of challenges, released daily from December 1st to 25th.

## Languages

Solutions may use various languages depending on what fits best:
- Python (procedural tasks, data processing)
- Prolog (logic puzzles, constraint problems)
- Others as appropriate

See individual year directories for specific solutions and running instructions.
